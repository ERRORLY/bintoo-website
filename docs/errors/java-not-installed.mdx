# Java Not Installed

Java is required to run any Minecraft server. If Java is not installed, or if you are using an unsupported version, the server may fail to start.

We recommend using **Java 21** or **Java 25**, as these versions are compatible with most modern Minecraft servers.

You can download the latest Java releases from Oracle here:

https://www.oracle.com/java/technologies/downloads/#java25
