# Adding Mods In Server
Bintoo includes built-in mod support, allowing you to download and install mods directly from services like **Modrinth** without leaving the app.

## Installing Mod Manually
If you'd like to install a mod from another source instead, you can do so manually:

1. Open **Bintoo** and go to the **Mods** section.
2. Click **Open Mod Folder**.
3. Copy or move the downloaded `.jar` mod file into the opened folder.
4. Re Launch the Server, and the mod will be loaded automatically (provided it's compatible with your game version and mod loader).

That's it! Your manually downloaded mods will appear alongside any mods installed through Bintoo.
