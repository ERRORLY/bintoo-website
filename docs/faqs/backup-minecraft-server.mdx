# Back Up a Minecraft Server

As of **Bintoo v1.0**, there is no built-in option to back up or export a server. This feature is planned for a future update.

## Manual Backup

You can still create a backup manually by accessing your server files.

### Steps

1. Open **Bintoo** and go to **Server Settings**.
2. Under the **Environment** section, click **Open Folder Path**.
3. Your system's file manager will open the server directory.

To back up your server, simply copy the files or folders you want to keep (such as your `world` folder, plugins, mods, or configuration files) to another location on your computer or an external drive.


:::danger
Note: Do not delete or modify files inside the Bintoo server directory unless you know what you are doing. Removing or changing files may corrupt the server or cause Bintoo to lose access to it.
:::

## Future Support

We are currently working on **Import** and **Export** features for a future version of Bintoo, making server backups and restoration much easier without requiring manual file management.
