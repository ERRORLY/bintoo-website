# Smart Screen Warning

If you are using Windows, you may see a blue security warning when downloading or opening **Bintoo**. This happens because the application is **not yet digitally signed with code-signing certificate** because of [lack of money](https://www.sectigo.com/ssl-certificates-tls/code-signing) :(

you can click `more info > run anyway` to avoid this issue

Windows displays this warning for unsigned applications to help protect users from potentially unsafe software. It does **not necessarily mean that Bintoo is harmful**, only that it has not been verified with a signing certificate.


We're a small indie project, and code-signing certificates is expensive. We hope to sign future releases of Bintoo once the project grows or we're able to afford one. If you'd like to help support the project and make signed releases possible, consider making a donation.

Thank you for supporting Bintoo!
